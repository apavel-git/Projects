В файле находятся примеры SQL запросов для проекта по анализу данных игрового проекта

### Цель проекта: 
изучить влияние характеристик игроков и их игровых персонажей 
на покупку внутриигровой валюты, а также оценить активность игроков при совершении 
внутриигровых покупок

### Часть 1. Исследовательский анализ данных
Задача: Команде игры важно знать, какая доля игроков покупает внутриигровую валюту за реальные деньги, 
то есть долю платящих игроков. Их также интересует, как игроки совершают внутриигровые покупки.

### Часть 2. Решение ad hoc задач
Задача: Анализ зависимости активности игроков от расы персонажа
