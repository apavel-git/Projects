В этой тетрадке показан пример настройки автоматической обработки данных при помощи DAG в AirFlow.

**Цель:** Создать витрину данных, которая будет автоматически агрегировать информацию о поездках по каждому способу оплаты.

**Содержание тетрадки:**
1. Настройка Spark-агрегации

    Расчет количества поездок, средней стоимости поездки, средних чаевых и суммарной выручки по каждому типу оплаты.
2. Настройка DAG

    DAG ежедневно:
   - Проверяет наличие новых файлов с данными;
   - Запускает Spark-задачу;
   - Формирует обновлённую итоговую таблицу.
