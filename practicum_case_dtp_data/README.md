# Анализ данных о ДТП

## Описание проекта

В этом проекте мы работаем с данными некоммерческого проекта, посвящённого проблеме дорожно-транспортных происшествий в России. Необходимо провести исследование данных и изучить взаимосвязи в них. Особенно важно узнать, с чем связана аварийность на дорогах.

## Задачи проекта

1. Предобработка данных: изменить названия столбцов и типы данных.
2. Ответить на вопросы заказчика.

## Вопросы заказчика

1. Как распределено количество участников ДТП и почему?
2. Между какими столбцами в датасете высокая корреляция?
3. Как связаны категории аварий и погодные условия?
4. Какова процентная разбивка аварий по видам освещённости с учетом пола участника?
5. Чем отличаются аварии без пострадавших от тех, в которых был один пострадавший или более?
6. Какие рекомендации вы можете дать заказчику относительно корреляций различных факторов с аварийностью?
