В данном файле представлен цикл работы с БД на SQL:

### Проектирование (DDL):
Создание структуры таблиц, настройка связей (Foreign Keys) и ограничений для обеспечения целостности данных.

### Манипуляция данными (DML):
Примеры наполнения, обновления и очистки таблиц.

### Оптимизация: 
Применение различных типов индексов (B-Tree, Hash) и партиционирование (Partitioning by Range) для ускорения выполнения запросов.

### Аналитика: 
Создание итоговой витрины данных (player_activity_vitrine).

### Анализ производительности: 
Использование EXPLAIN ANALYZE для проверки эффективности созданных индексов и структуры запросов.
